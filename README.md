# Happy Birthday Moon ❤️

A fully interactive, multi-page birthday surprise website built with HTML, CSS, and JavaScript.

## 🌟 Project Structure

```
project/
│
├── index.html          ← Main HTML (open this to start)
├── css/
│   └── style.css       ← All styles, animations, glassmorphism
├── js/
│   └── script.js       ← All JavaScript logic & interactions
├── music/
│   └── birthday.mp3    ← Birthday song (add your own MP3 here)
├── images/
│   ├── photo1.jpg      ← Memory gallery photos (replace with real ones)
│   ├── photo2.jpg
│   ├── photo3.jpg
│   ├── photo4.jpg
│   ├── photo5.jpg
│   └── photo6.jpg
└── README.md
```

## 🎉 Pages

| Page | Description |
|------|-------------|
| **Page 1** | Welcome page with animated stars, balloons, hearts, particles |
| **Page 2** | Calendar countdown June 1–15 with page-flip animation |
| **Page 3** | Birthday reveal popup with fireworks & confetti |
| **Page 4** | 3D birthday cake with flickering candles & music player |
| **Page 5** | Cake cutting animation with balloon rise & fireworks |
| **Page 6** | Memory gallery — masonry layout, hover zoom, lightbox |
| **Page 7** | Animated typing love message with floating flowers |

## 🚀 How to Run

Simply open `index.html` in any modern browser. No server required!

```
Double-click → index.html
```

## 🎵 Adding Birthday Music

1. Download a birthday MP3 file
2. Rename it to `birthday.mp3`
3. Place it in the `music/` folder

Free sources: [Free Music Archive](https://freemusicarchive.org), [Pixabay Music](https://pixabay.com/music/)

## 📸 Replacing Photos

Replace the placeholder images in the `images/` folder:
- `photo1.jpg` through `photo6.jpg`
- Any size works — the gallery handles it automatically

## 🎨 Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Glassmorphism, animations, responsive design
- **Vanilla JavaScript** — All interactivity
- **GSAP** — Advanced animations
- **Canvas Confetti** — Confetti explosions
- **Google Fonts** — Dancing Script, Playfair Display, Poppins
- **Animate.css** — Entry animations
- **Canvas API** — Stars, fireworks, particles
